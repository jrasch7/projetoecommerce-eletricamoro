# SaaS E-commerce & Gestão

Uma plataforma SaaS completa e totalmente personalizável para e-commerce e gestão integrada, focada no setor de materiais elétricos. Desenvolvida para oferecer alta performance, flexibilidade de catálogo e inteligência de dados.

## 🚀 Tecnologias Utilizadas

- **Backend:** Node.js com TypeScript

- **Banco de Dados:** PostgreSQL (Relacional e Robusto)

- **ORM:** Prisma (Gestão de Schema e Migrations)

- **Segurança:** Helmet (Headers HTTP) & Express-Rate-Limit (Proteção de API)

- **Frontend:** HTML5, JS Vanilla e Tailwind CSS (Design Moderno e Responsivo)

- **BI & Analytics:** Chart.js para dashboards dinâmicos

- **Interface:** Phosphor Icons

## 📋 Funcionalidades de Gestão & Venda

- **Catálogo Flexível:** Gestão de produtos com suporte a múltiplas imagens, especificações técnicas (voltagem, modelo, marca) e controle de estoque.

- **Hierarquia Dinâmica:** Categorias e Subcategorias totalmente customizáveis que protegem a integridade dos produtos.

- **Painel Administrativo (BI):** Dashboard integrado para monitoramento de vendas, fluxo de pedidos e métricas de desempenho.

- **Arquitetura Resiliente:** Sistema preparado para edições rápidas e desvínculos de categorias sem perda de dados históricos.
